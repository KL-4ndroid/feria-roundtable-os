# Archive｜封存區

Use this folder for inactive, retired, superseded, or raw historical material.

---

## What belongs here

- retired project folders
- superseded versions
- raw logs that no longer need to stay in active project files
- old templates
- cancelled system experiments

---

## What should not be archived too early

Do not archive content that is still needed for active decision-making.

Before archiving, make sure:

- the active project brief is updated
- the compacted summary is updated
- decisions are recorded
- risks and objections are preserved
- raw content remains traceable

---

## Naming convention

```text
YYYY-MM-DD_description
```

or

```text
PROJECT-ID_archived-reason
```

---

## Archive rule

Archiving is not deletion.

Archived files should remain available for traceability, but should not be treated as the current source of truth.
