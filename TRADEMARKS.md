# Trademark and Branding Policy

This file explains how the Roundtable OS name and branding may be used.

---

## License and branding are separate

The materials in this repository are open under CC BY 4.0 unless otherwise stated.

That license lets people use, share, adapt, and build upon the documentation, templates, and workflows with attribution.

However, the license does not automatically grant rights to use official names, marks, logos, badges, or branding in a way that suggests an official connection.

---

## Names covered by this policy

This policy applies to names and marks such as:

- Roundtable OS
- 圓桌工作系統
- Roundtable OS Core
- Roundtable OS Official
- Roundtable OS Pro
- Roundtable OS Certified
- any future official logos, badges, icons, or certification marks

---

## Allowed uses

You may use the name Roundtable OS to:

- give attribution
- describe that your work is based on Roundtable OS
- link back to the original repository
- discuss, review, teach, or explain the system
- identify compatibility with Roundtable OS

Examples:

```text
Based on Roundtable OS｜圓桌工作系統.
```

```text
A modified workflow inspired by Roundtable OS.
```

```text
Compatible with the Roundtable OS project-folder structure.
```

---

## Not allowed without permission

Please do not use Roundtable OS branding in a way that suggests your project is the official version.

Avoid names such as:

- Official Roundtable OS
- Roundtable OS Certified
- Roundtable OS Pro
- Roundtable OS Academy
- Roundtable OS Consultant
- Roundtable OS Partner

unless permission has been granted by the project maintainer.

---

## Modified versions

If you publish a modified version, please make it clear that it is modified.

Recommended wording:

```text
Adapted from Roundtable OS｜圓桌工作系統. This version has been modified and is not the official Roundtable OS project.
```

---

## Why this policy exists

The goal is to keep the core system open while protecting clarity for users.

People should be able to freely use and adapt the system, while still being able to identify the official source and avoid confusion between official and modified versions.
