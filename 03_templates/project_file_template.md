# Project File Template｜專案檔模板

Use this template when creating a new project folder.

---

## Recommended folder structure

```text
PROJECT-ID_project-name/
├─ 00_project_brief.md
├─ 01_roundtable_log.md
├─ 02_role_outputs/
│  ├─ 00_control_room.md
│  ├─ 01_audience_insight.md
│  ├─ 02_shortform.md
│  ├─ 03_content.md
│  ├─ 04_free_tool.md
│  ├─ 05_template_product.md
│  ├─ 06_product.md
│  └─ 07_compaction.md
├─ 03_decisions.md
├─ 04_tasks.md
├─ 05_handoffs.md
├─ 06_assets.md
├─ 07_compacted_summary.md
├─ 08_changelog.md
└─ 99_raw_archive.md
```

---

## Minimum required files

For a lightweight project, use only:

```text
00_project_brief.md
01_roundtable_log.md
03_decisions.md
07_compacted_summary.md
```

Add other files when the project becomes more complex.

---

## Creation checklist

- [ ] Create project folder
- [ ] Fill `00_project_brief.md`
- [ ] Add project to `00_system/project_status_index.md`
- [ ] Add project to the relevant global index
- [ ] Add first role response to `01_roundtable_log.md`
- [ ] Create first compacted summary when needed
