# Féria Role Settings｜角色聊天室設定

這個資料夾放置 Féria｜出攤筆記 Roundtable OS 的 8 個角色聊天室設定。

使用方式：

1. 在 ChatGPT Project 中建立對應角色聊天室。
2. 將對應 `.md` 檔案內容貼到該聊天室的第一則訊息，或作為該聊天室的角色設定參考。
3. 每個角色只負責自己的職責範圍。
4. 若任務超出角色職責，請交給對應角色或總控室。
5. 多角色產出後，回到總控室收斂決策。
6. 文件過長、重複或脈絡混亂時，交給文件減肥室整理。

---

## 角色清單

| 編號 | 角色聊天室 | 用途 |
|---|---|---|
| 00 | 總控室｜策略與決策中心 | 統整、決策、優先順序、避免發散 |
| 01 | 受眾洞察室｜小品牌痛點研究員 | 痛點、客群、需求真實性分析 |
| 02 | Thread 文字口碑室｜短文觀察企劃 | Threads 短文、觀察、討論型內容 |
| 03 | Instagram 內容室｜收藏型內容企劃 | IG 輪播、限動、CTA、導流內容 |
| 04 | 免費工具產品室｜小品程式企劃 | 免費工具構想、欄位、輸出、MVP |
| 05 | 模板商品室｜知識產品設計師 | 付費模板、SOP、文案包、銷售頁 |
| 06 | APP 產品沉澱室｜Feria 產品經理 | 功能假設、MVP 判斷、產品化節奏 |
| 07 | 文件減肥室｜脈絡整理與不失真壓縮員 | 文件壓縮、脈絡整理、保留決策與風險 |

---

## 建議聊天室建立順序

1. 00_總控室
2. 07_文件減肥室
3. 01_受眾洞察室
4. 02_Thread文字口碑室
5. 03_Instagram內容室
6. 04_免費工具產品室
7. 05_模板商品室
8. 06_APP產品沉澱室

總控室與文件減肥室建議最先建立，因為它們負責維持整套系統不發散、不臃腫。
